# Class: RSAPublicKey

Represents an RSA key as defined in RFC 8017 (PKCS #1 v2.2).

## Asn

```asn
RSAPublicKey ::= SEQUENCE {
    modulus           INTEGER,  -- n
    publicExponent    INTEGER   -- e
}
```
