# Function: uint8ArrayToHex()

> **uint8ArrayToHex**(`bytes`, `prefix`): `string`

### bytes

`Uint8Array`\<`ArrayBuffer`\>

The Uint8Array<ArrayBuffer> to convert

### prefix

`boolean` = `false`

Whether to include the '0x' prefix (default: false)

## Returns

`string`
